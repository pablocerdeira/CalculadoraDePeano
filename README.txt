Código fonte de uma calculadora que realiza operações aritiméticas com base nos axiomas de Peano.

Ou seja, ela não calcula 1 + 2 = 3 como se fossem inteiros, mas sim utiliza os axiomas de Peano para entender que 1 é um sucessor de 0, que 2 é o sucessor do sucessor de 0 e assim realiza a soma localizando o sucessor do sucessor do sucessor de 0, ou seja, 3.

Com isso, essa calculadora pode utilizar qualquer definição de elementos para o conjunto dos naturais. Ela pode somar utilizando 0,1,2,3,…,9, ou a,b,c,…,z, ou os naipes do baralho etc.

O objetivo deste programa é demonstrar a utilização dos axiomas de Peano na prática.
